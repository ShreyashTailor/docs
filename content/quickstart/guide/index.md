To get your own <mark>is-an-ai.dev</mark> subdomain, first check its availability at [here](https://is-an-ai.dev/).

# How to Register

<ol>
If a subdomain you want to claim is available, you can register it by the following the below steps:

<li>

Fork our github [repository](https://github.com/is-an-ai-dev/register).

</li>

<li>

Create a new file named <mark>your-subdomain.json</mark> in <mark>domains/</mark> directory following all the valid rules listed in [Request Format](https://example.com)

</li>
<li>

Create a PR and tick the checkboxes to agree to our conditions.

</li>
<li>

Patiently wait while a maintainer or a reviewer verifies your PR and merge it. Make sure to keep an eye on the PR as questions may be asked regarding your use case

</li>

</ol>
